# Glass Jungle

## Controls
- **W, A, S, D:** Move
- **F:** Stab
- **Space:** Jump
- **Control:** Crouch
- **Mouse movement:** Control camera
- **Mouse left button:** Shoot
- **Mouse right button:** Aim
